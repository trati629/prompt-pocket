Import Format Guide

  {
    "title":    "My Template",
    "body":     "Explain {{topic}} simply.",
    "category": "general",

Field aliases
title also accepted: name body also accepted: prompt, content, text category also accepted: cat, type tags also accepted: tag, labels, keywords 
title,body,category,tags
"Code Review","Review {{language}} code.","developer","review,code"
"Quick Summary","Summarise {{text}}.","general",""
  • Header row required (case-insensitive, aliases accepted)
  • Wrap values containing commas in double quotes
  • tags cell: comma-separated values inside quotes
  • File encoding: UTF-8
